Instructions d'authentification Udemy

Pour vous connecter avec Udemy :

  1. Une popup s'ouvrira vers la page de connexion Udemy
  2. Connectez-vous normalement avec vos identifiants Udemy
  3. Une fois connecté, l'application détectera automatiquement votre token
  4. La popup se fermera et vous serez connecté à l'application

Note : Assurez-vous d'autoriser les popups pour ce site.
